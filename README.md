# Velvet AI - The AI Website Builder

Velvet AI is a cutting-edge, AI-powered website builder that empowers users to generate, edit, and preview dynamic web applications through a conversational interface. Built with modern web technologies, it seamlessly blends natural language processing with a robust coding sandbox.

## 🚀 Key Features

- **Conversational Interface:** Create and iterate on website designs simply by chatting with the AI.
- **Integrated Code Sandbox:** Powered by [E2B](https://e2b.dev/), the app features a secure, sandboxed environment for real-time code execution and terminal access.
- **In-Browser Editor:** Embedded [Monaco Editor](https://microsoft.github.io/monaco-editor/) for advanced users to inspect and tweak the generated code directly in the browser.
- **Stunning UI & Animations:** Built with sleek micro-animations and smooth scrolling, utilizing [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/), and [Lenis](https://lenis.studiofreight.com/).
- **3D Graphics Integration:** Ready for immersive experiences with [Three.js](https://threejs.org/).

## 💻 Tech Stack

- **Framework:** Next.js 15+ & React 19
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion, GSAP, Lenis
- **3D Rendering:** Three.js
- **Code Editing:** @monaco-editor/react
- **Execution Environment:** E2B
- **Icons:** Lucide React

## 🛠️ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Divyezh/Velvet-AI---The-Ai-website-Builder.git
   ```
2. Navigate into the application directory:
   ```bash
   cd "Velvet-AI---The-Ai-website-Builder/ai-builder"
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Configuration

Create a `.env` file in the `ai-builder` directory and add your necessary environment variables (e.g., your AI provider keys, E2B API key).

### Running the Development Server

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
