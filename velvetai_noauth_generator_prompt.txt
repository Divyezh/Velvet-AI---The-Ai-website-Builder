  - The HTML must be complete and render perfectly in an iframe
  ---

  User message: `Build a complete website for: ${prompt}`

  Stream the response back using Next.js streaming:
    Return new Response with ReadableStream
    Use client.messages.stream() for streaming
    Pipe text chunks directly to the stream
    On stream end, close the ReadableStream controller

  Error handling:
    If ANTHROPIC_API_KEY missing: return 500 { error: "API key not configured" }
    If prompt empty or < 5 chars: return 400 { error: "Prompt too short" }
    If prompt > 1000 chars: return 400 { error: "Prompt too long" }
    Wrap in try/catch, return 500 on any Anthropic error

handleGenerate function:
  1. setIsGenerating(true), setHtml(""), setElapsedTime(0)
  2. Start timer interval (setInterval every 1000ms incrementing elapsedTime)
  3. POST to /api/generate with { prompt }
  4. Read response as a stream using response.body.getReader()
  5. Decode chunks with TextDecoder
  6. Append each chunk to html state: setHtml(prev => prev + chunk)
  7. React re-renders iframe srcdoc in real time as HTML streams in
  8. On stream complete: clearInterval, setIsGenerating(false)
  9. On error: show error toast, setIsGenerating(false)
